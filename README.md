# 🎾 내일은 서브왕 — 서브 셀프 코치

**https://hunydah111.github.io/serve-king/**

코트에서 혼자 서브 연습할 때 켜두는 도구입니다.

- 서브 결과를 원탭 기록: 네트 / 길다 / 좌·우 와이드 / 얕음 / 굿 / 토스 캐치
- 자주 나오는 원인 그룹 + "다음 서브에선 이것 하나만" 큐 하나
- 세션 기록·패턴 진단 (기기 저장, 서버 전송 없음)
- 🔬 서브 시뮬레이터: 토스·타점·스핀에 따른 궤적 (공기저항+마그누스 수치적분, 추정값)
- 📸 굿 서브는 카드로 저장·공유

카메라 없이 결과만으로 추정하는 일반론이며 코치 레슨을 대체하지 않습니다.
